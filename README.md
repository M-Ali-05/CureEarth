# Cure Earth: Global Defense

Cure Earth: Global Defense is a strategy game built with Python and Pygame.

In this game, the player becomes the leader of a global crisis response center and must stop a pandemic from spreading across the world.

## Main Idea
Unlike games where the player spreads disease, this project focuses on saving humanity and stabilizing the planet.

The player must manage six major world regions:
- North America
- South America
- Europe
- Africa
- Asia
- Australia

Each region has its own indicators:
- infection
- ecology
- economy
- panic
- medical system
- public trust

The disease spreads through air and sea routes, so the player must make strategic decisions and balance health, economy, ecology, and transportation.

## Features
- World map with six regions
- Pandemic spread system
- Air and sea transportation routes
- Region-based actions
- Global projects
- Difficulty levels
- Victory and defeat screens
- Animated airplanes and ships
- Country points inside regions

## Technologies Used
- Python
- Pygame
- PyCharm
